A propos

J'ai effectué une formation Openclassrooms consacré au développement Fullstack, avec les frameworks Angular et Java. Cette formation m'a permis de gérer le développement d'une application complète, de la conception jusqu'à l'intégration et au déploiement continu. Pour le frontend, j'ai notamment appris les formulaires réactifs, les Observables et les opérateurs de haut ou bas niveau, les Guards, Interceptors, et le Lazy-Loading. Le backend m'a permis de connaître l'architecture en couche (Dto-Controller-Service-Repository), les protocoles de sécurité (Spring Security) Pour chacun de ces frameworks, une suite de tests associée m'a permis d'assuré la stabilité des applications.

  • Angular
  • Java Spring
  • Spring Security
  • Outils de conception (Figma, Lucidchart)
  • Tests (frontend, backend, end-to-end)
  • Développement et Intégration continues (Github Actions)
  • Analyse de qualité (Sonarqube)
  • Base de données (MCD, MLD, formes normales)

Au-delà de l'aspect technique, j'ai également appris à gérer la conception d'une application et à communiquer les informations importantes au sein de l'équipe de développement. Je peux réaliser un cahier des charges, un document d'architecture, une matrice de risques.
